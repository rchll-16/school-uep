# 🎓 School Website — **School‑UEP**

A clean, responsive, and informative school website built using **HTML5** and **CSS3**.  
Ideal for institutional presentation, this static site includes pages for home, about, academic programs, events, and contact details.

## 📖 Project Overview

**`school-uep`** is a static, multi-page website designed for school-related information delivery.  
Key pages included:
- **Home (`index.html`)** – Welcoming landing page  
- **About (`about.html`)** – Background or mission of the school  
- **Academic Programs (`acad-prog.html`)** – List of courses or academic offerings  
- **Events (`events.html`)** – School-based events or announcements  
- **Contacts (`contacts.html`)** – Contact details or inquiry form  
- **`style.css`** – Central stylesheet for consistent design

## 🚀 Setup & Viewing

1. **Clone the repository**
   \`\`\`bash
   git clone https://github.com/rchll-16/school-uep.git
   cd school-uep
   \`\`\`

2. **Open locally**
   Simply double-click any `.html` file (e.g. `index.html`) to view in your browser.

3. **Enhancement Ideas**
